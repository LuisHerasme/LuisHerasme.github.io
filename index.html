Big changes in FisicaJs

In the new FisicaJs there will be collision modes:

  1. SCC (Simple Circle Collision)
  2. AABB (Axis-Aligned Bounding Box)
  3. Mesh collider
they are organized from faster to slower, the collision mode will allow me to make the physics engine run faster if you set certain collision type


You can activate each different mode by typing 


const fisica = require('fisica')
fisica.mode('SCC')
